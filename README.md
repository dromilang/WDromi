# WDromi
CDN Per avviare Dromi nel Web!

---

## Usare la libreria

Aggiungi questo al tuo JS:

```javascript
import { runDromi } from "https://dromilang.github.io/WDromi/dromi-v1.1/wdromi.js";
```

In questo JS ora potrai usare `runDromi` (ricordati di inserire `type="module"` nelle proprietà dello `script`!)

---

## Versioni supportate

WDromi supporta le seguenti versioni di Dromi:

- v1.1.x (`dromi-v1.1/wdromi.js`)
